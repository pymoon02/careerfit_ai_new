# CareerFit AI

> 취업·공모전 데이터 기반 맞춤형 AI 포트폴리오 코치



## 프로젝트 개요



Q1) 취업 또는 공모전을 준비하면서 어려웠던 점은?
- 어떤 스킬을 준비해야 할지 모른다.
- 어느 정도 수준으로 준비해야 하는지 모르겠다.



## 기술 스택



| 영역 | 기술 |

|---|---|

| 백엔드 | Python, FastAPI |

| AI API | Gemini 2.5 Flash-Lite |

| 데이터 | Pandas, SQLite, ChromaDB |

| 프론트엔드 | React, Vite |

| 실행 환경 | Docker |
