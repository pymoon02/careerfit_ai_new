# 개인 회고 — CareerFit AI 부트캠프 5일



## 오늘 날짜

2026.07.07



## 1. 5일 동안 가장 인상 깊었던 경험 1가지

RAG 기반 AI 서비스를 직접 구현한 것



## 2. 가장 오래 막혔던 구간

막힌 구간: Docker를 이용해 프로젝트를 컨테이너에서 실행하고 환경변수를 전달하는 과정에서 가장 오래 어려움을 겪었습니다.

원인: Docker 명령어 사용법이 익숙하지 않았고, Docker Desktop 실행 여부, 컨테이너 실행 방식 등을 제대로 이해하지 못했습니다.

해결 방법: 오류 메시지를 하나씩 확인하면서 원인을 분석했고, Docker Desktop 실행 상태를 확인하는 방법과 docker build, docker run, .env 환경변수 전달 방식을 익혔습니다.



## 3. 처음으로 "이게 되는구나" 싶었던 순간

React 화면에서 입력한 정보를 FastAPI 서버로 전달하고, Gemini API가 생성한 분석 결과가 다시 화면에 출력되는 것을 처음 확인했을 때 가장 큰 성취감을 느꼈습니다.



## 4. 내 서비스의 아쉬운 점

추천 기준이 비교적 단순하여 사용자 이력서나 관심 분야를 더욱 세밀하게 분석하는 기능은 구현하지 못했습니다.



## 5. 다음에 혼자 도전해보고 싶은 것

실제 취업 공고 API를 연동하여 데이터를 자동으로 수집하는 기능을 구현해 보고 싶습니다.
또한 이력서 PDF를 업로드하면 내용을 분석하여 부족한 역량을 추천하는 기능과 사용자별 맞춤 학습 로드맵을 제공하는 기능도 추가해 보고 싶습니다.



## 6. 이 경험을 포트폴리오에서 어떻게 설명할 것인가

취업 공고 데이터를 Pandas로 전처리하여 SQLite와 ChromaDB에 저장하고, RAG 기반 검색과 Gemini API를 활용해 사용자 맞춤형 역량 분석 서비스를 구현했습니다. React와 FastAPI를 연동하여 웹 서비스를 개발하고 Docker를 이용해 실행 환경을 컨테이너화했으며 Render를 통해 배포 가능한 형태로 프로젝트를 완성했습니다.



한 줄 요약:

취업 공고 데이터를 전처리하고 RAG 기반 AI 분석 서비스를 FastAPI와 React로 구현했으며, Docker와 Render를 활용해 배포 가능한 웹 서비스로 완성했습니다.
